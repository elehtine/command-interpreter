# Command interpreter

Cli for interpreting commands.

## Usage

```
python3 main.py 
> add
Give two integers
first: one
Invalid number: one
first: 1
second: 2
1 + 2 = 3
> 
Empty line: exit
```

## Test

Run following commands to execute tests:

```
python3 test_add.py
python3 test_empty.py
python3 test_invalid.py
```
